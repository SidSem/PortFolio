# Siddharth Semwal Portfolio

A modern and interactive **3D Developer Portfolio** built with **React**, **Three.js**, **React Three Fiber**, and **Tailwind CSS**.

Designed to showcase education, technical skills, projects, and provide a functional contact experience through EmailJS.

---

## ✨ Features

* Interactive 3D Desktop Scene
* Animated Hero Section
* Education Timeline
* Technology Spheres
* Responsive Design
* Framer Motion Animations
* Project Showcase Section
* Functional Contact Form with EmailJS
* Smooth Scrolling Navigation
* Mobile Friendly Layout

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file.

Add:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id

VITE_EMAILJS_TEMPLATE_ID=your_template_id

VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
